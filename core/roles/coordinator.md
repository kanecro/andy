# Role: coordinator

## Purpose

Plan, decompose, delegate, integrate, and escalate.

## Capability profile

- planning: high
- code_editing: false by default
- review: medium
- context_management: high

## Responsibilities

- Read user goal and high-level artifacts
- Choose workflows and roles
- Assign disjoint write scopes
- Avoid unnecessary implementation detail in coordinator context
- Run final verification and summarize evidence
